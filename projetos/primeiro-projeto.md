# Primeiro Projeto Versionado

Este arquivo representa um exemplo de projeto versionado com Git e GitHub.

## Descrição

O objetivo deste projeto é registrar a prática de criação de arquivos, organização de pastas, escrita de documentação e versionamento básico.

## Tecnologias utilizadas

- Git
- GitHub
- Markdown

## Etapas realizadas

1. Criação do repositório no GitHub
2. Edição do arquivo README.md
3. Criação da pasta de anotações
4. Criação da pasta de projetos
5. Criação da pasta de imagens
6. Registro das alterações por meio de commits

## O que foi aprendido

Durante este projeto, foram praticados conceitos essenciais de versionamento, como repositório, commit, branch, documentação e organização de arquivos.

Também foi possível entender como o GitHub pode ser utilizado como portfólio profissional, permitindo demonstrar projetos, estudos e evolução técnica.

## Conclusão

O Git e o GitHub são ferramentas fundamentais para profissionais de tecnologia, pois permitem controlar versões, colaborar em projetos e manter um histórico organizado das alterações realizadas.

Este desafio contribuiu para consolidar os primeiros passos no uso dessas ferramentas.
