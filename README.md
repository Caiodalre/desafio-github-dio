# Desafio Git/GitHub - DIO

Repositório criado para o desafio de projeto da DIO sobre Git e GitHub.

## Objetivo

Este projeto tem como objetivo praticar os principais conceitos de versionamento de código utilizando Git e GitHub.

## Conteúdos praticados

- Criação de repositório no GitHub
- Versionamento de arquivos
- Uso de Markdown
- Organização de projeto
- Registro de anotações técnicas
- Publicação de projeto no GitHub
